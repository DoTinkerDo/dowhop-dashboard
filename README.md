# dowhop-me
For everything on DoWhop related to a single user: my DoWhops, my Profile, my Settings, my Notifications, my DoneWhops

___
### Usage:
```sh

Local development
$ yarn
$ $ yarn start or $ NODE_ENV=production yarn start

$ yarn flow

Testing
$ yarn test

Deploy to Firebase
$ yarn build
$ yarn deploy

```
